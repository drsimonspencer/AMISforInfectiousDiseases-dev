Fitting ascaris data

This vignette demonstrates the AMISforInfectiousDiseases package through a case study considering Ascariasis in Ethiopia. Ascariasis is one of the most common worm infections in people worldwide. This disease is caused by Ascaris lumbricoides, also known as roundworms, which are an intestinal nematode. This was chosen as an example to illustrate the following package features: modelling from a single prevalence map, enforcing boundary conditions on model parameters, and post-processing of the results to produce forward projections of prevalence using sub-national estimates of the model parameters.

Prevalence map

Ascaris surveys for Ethiopia are available upon request from the Global Atlas of Helminth Infection developed by the London Applied & Spatial Epidemiology Research Group (LASER) (Rachel L et al. 2014). (Retkute et al. 2021) uses these surveys to obtain national level prevalence maps, which will be used in this analysis. These maps consist of \(M=100\) samples of a prevalence map for Ascaris in Ethiopia with \(11369\) locations at 10km \(\times\) 10km resolution.

The first two columns of the ascaris data frame are the geographical coordinates (longitude and latitude). The third column contains the observed prevalence in the location (row). The remaining columns are the \(M=100\) samples from a geostatistical map.

Transmission model

The transmission model for Ascaris lumbricoides can be described by the mathematical relationship between prevalence and the mean intensity of infection. From (Retkute et al. 2021), the transmission model for Ascaris uses the negative binomial model where \(W>0\) is the mean number of worms per host and \(k\) is the worm clumping coefficient, \[\begin{equation} P = 1 - \left( 1 + \frac{W}{k} \right)^{-k} \end{equation}\]

In this example, the natural logarithm of the model parameter \(W\) is sampled. Thus the transmission model passed to function amis is given by

ascaris_model <- function(seeds, params, n_tims=1){
  p <- matrix(NA, nrow=nrow(params), ncol=n_tims)
  for(t in 1:n_tims){
    W <- exp(params[,1])
    k <- params[,2]
    p[,t] <- 1-(1+W/k)^(-k)
  }
  return(p)
}

Priors

The priors from (Retkute et al. 2021) for the model parameters are borrowed in this example. These priors reflect what is currently known about the interaction between the worm burden and clumping coefficient based on the literature. In summary, a uniform prior is placed on the log worm burden and the relationship between the clumping coefficient and worm burden is estimated through a linear regression. Formally this is

\[\begin{align} \log(W) &\sim \mathcal{U}\bigg(\log(0.01),\log(60)\bigg) \\ k | W &\sim \mathcal{N}\bigg(\hat{\alpha}+ \hat{\beta} \times W,\sigma(W)^2\bigg)\\ \end{align}\]

where \(\hat{\alpha}\) and \(\hat{\beta}\) are the intercept and slope coefficients respectively for the regression line of worm burden against the clumping coefficient, and \(\sigma(W)\) is \(5\) times the standard error from this analysis. Refer to (Retkute et al. 2021) for further details on the prior selection.

The prior object passed to is given by

fit.v <- c(0.33371009, 0.01719462)
fit.inflation.factor <- 5
fit.hess<-matrix(0, nrow = 2, ncol = 2)
fit.hess[[1,1]] <- 5138.97
fit.hess[[1,2]] <- 49499.4
fit.hess[[2,1]] <- 49499.4
fit.hess[[2,2]] <- 677831

rprior <- function(n) {
  params <- matrix(NA,n,2)
  colnames(params) <- c("logW","k")
  params[,1] <- runif(n,log(0.01),log(60))
  params[,2] <- sapply(1:n, function(i) {
    rnorm(1, mean = fit.v[1] + fit.v[2]*exp(params[i,1]), 
          sd = fit.inflation.factor*sqrt(c(1,exp(params[i,1]))%*%solve(fit.hess,c(1,exp(params[i,1])))))
  })
  return(params)
}

dprior <- function(x, log=FALSE) {
  if (log) {
    dlogW <- dunif(x[1],log(0.01),log(60),log=T)
    dk <- dnorm(x[2], mean = fit.v[1] + fit.v[2]*exp(x[1]), 
               sd = fit.inflation.factor*sqrt(c(1,exp(x[1]))%*%solve(fit.hess,c(1,exp(x[1])))),log=T) 
    return(sum(dlogW,dk))
  } else {
    dlogW <- dunif(x[1],log(0.01),log(60))
    dk <- dnorm(x[2], mean = fit.v[1] + fit.v[2]*exp(x[1]), 
               sd = fit.inflation.factor*sqrt(c(1,exp(x[1]))%*%solve(fit.hess,c(1,exp(x[1]))))) 
    return(prod(dlogW,dk))
  }
}
ascaris_prior <- list(rprior=rprior, dprior=dprior)

Running amis

Default amis settings (with parameter boundaries)

amis_params <- AMISforInfectiousDiseases:::default_amis_params()
boundaries_param <- matrix(NA, 2, 2)  
boundaries_param[1,] <- c(-Inf, Inf)
boundaries_param[2,] <- c(0, Inf)
amis_params$boundaries_param <- boundaries_param

Note that instead of defining boundaries through the object amis_params, the user could alternatively set the prevalence to NA directly in the transmission model when \(k\) is negative and thus out of bounds.

The boundaries for prevalence values, which are in \([0,1]\), are taken into account by default (through amis_params$boundaries).

This example instead uses the Gaussian kernel for density estimation. To define this in the package, one needs to define sigma in the object amis_params.

amis_params$sigma <- 0.1
set.seed(123)
out <- amis(ascaris_map, ascaris_model, ascaris_prior, amis_params)

Forward projection

To produce a projection of future prevalence under different treatment scenarios, for each pixel 100 parameter sets are sampled from the weighted set of parameters obtained from out (i.e. using default amis settings). The sub-sampled model parameters are then fed into the transmission model.

sampled_pars <- sample_parameters(x=out, n_samples=100, locations=1:nrow(ascaris_map))

ascaris_model_with_treatment <- function(params, n_treat_years){
  W <- exp(params[,"logW"]) * 0.9^n_treat_years
  k <- params[,"k"]
  p <- 1-(1+W/k)^(-k)
  return(p)
}
prev_after_treatment <- ascaris_model_with_treatment(params = sampled_pars, n_treat_years = 5)
sampled_pars <- cbind(sampled_pars, prev_after_treatment)
treat_res <- aggregate(prev_after_treatment~location, data=sampled_pars, FUN="median")
ascaris_proj_prev <- ascaris_obs_prev
ascaris_proj_prev$prevalence <- treat_res$prev_after_treatment
data_proj <- rbind(cbind(type="Precontrol",ascaris_obs_prev),
                  cbind(type="Postcontrol",ascaris_proj_prev))
data_proj$type <- factor(data_proj$type, levels=c("Precontrol","Postcontrol"))
data_proj_sf <- sf::st_as_sf(data_proj, coords = c("lon", "lat"))

ggplot(data=data_proj_sf) + 
  geom_sf(aes(colour=prevalence),shape=15,size=3) +
  facet_wrap(~type) +
  scale_colour_gradientn(colours=rev(viridis::magma(6)),
                         name="Prevalence")
